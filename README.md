Usage
---
Setup
---
Install dependencies with
```npm install```

Start the database with
```npm run database```

Start the server with
```npm start```

Development
---
Start the database
```npm run database```

Compile with Webpack
```npm run compile-dev```

Start the development server
```npm start-dev```
